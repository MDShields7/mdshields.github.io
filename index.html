Fetch The News
New Trivia Party Game
A Javascript and React.js web app with responsive interface: desktop for hosting games, and mobile for player controllers. Socket.IO is used for real-time gameplay across devices. Redux is used for state management. A normalized database is created with postgreSQL. Server side is done through Node.js and Express. Custom graphics are created with Adobe Illustrator. Styling was done with Sass. Git version control is used.
Moment
AirBnb Experiences Clone
A Javascript and React.js web app state management is done with Context API. Password are hashed through Bcrypt, and user state is stored with express-session. Reusable components are utilized, for containers and content. Search and filter features are sorted with Lodash. Dates are created and processed with Moment.js. Online payments are processed with Stripe. Project Management of group tasks are done with Zenhub. Database is planned with MySQLWorkbench.
